pine.io
=======

A simple, fast, general purpose, realtime, RESTful database-as-a-service.
A HTML 5 Application platform-as-a-service

Setup
-----

	npm -g install nodemon
	npm -g install coffee-script
	npm -g install mocha

	npm install

Run it!
-------

	NODE_ENV=development nodemon app.coffee


Testing
-------

	make test
