Fast

PineIO apps are fast because they are pure HTML. No server side code is ever executed. This forces developers to build their applications around web services. By doing so, scaling becomes cheap.

Simple

PineIO is simple because its just JSON. Easily work with JSON objects inside Javascript. We also index the database for you, so you don't have to.

Mobile

PineIO is great for mobile app, because its fast and simple. Mobile applications require a low footprint, Pine apps cache very well. Deploy your HTML 5 app to Apple's App Store with tools like PhoneGap.
