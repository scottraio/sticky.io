REST API

The REST API allows you access endpoints through HTTP.

Databases

Resource URL HTTP Verb Role
https://pine.io/databases GET Lists all databases
https://pine.io/databases POST Creates a new database
https://pine.io/databases/:title GET Find a database by name
https://pine.io/databases/:title PUT Update database attributes
https://pine.io/databases/:title DELETE Remove a database

Tables

Resource URL HTTP Verb Role
https://pine.io/:db_title/tables GET Lists all tables for a database
https://pine.io/:db_title/tables POST Creates a new table for a database
https://pine.io/:db_title/tables/:title GET Returns details of table
https://pine.io/:db_title/tables/:title PUT Updates a table's attributes
https://pine.io/:db_title/tables/:title DELETE Remove a table

Records

Resource URL HTTP Verb Role
https://pine.io/:db_title/records/:title/tables GET Lists all databases
