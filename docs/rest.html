REST API

The REST API allows you access endpoints through HTTP.

Databases

Resource URL HTTP Verb Role
https://pine.io/databases.json GET Lists all databases
https://pine.io/databases POST Creates a new database
https://pine.io/databases/:title.json GET Find a database by name
https://pine.io/databases/:title PUT Update database attributes
https://pine.io/databases/:title DELETE Remove a database

Collections

Resource URL HTTP Verb Role
https://pine.io/:db_title/collections.json GET Lists all collections for a database
https://pine.io/:db_title/collections POST Creates a new collection for a database
https://pine.io/:db_title/:title.json GET Returns details of collection
https://pine.io/:db_title/:title PUT Updates a collection's attributes
https://pine.io/:db_title/collections/:title DELETE Remove a collection

Records

Resource URL HTTP Verb Role
https://pine.io/:db_title/:collection_title/records.json GET Lists all records
https://pine.io/:db_title/:collection_title/records POST Creates a new record
https://pine.io/:db_title/:collection_title/records/:id GET Returns details of record
https://pine.io/:db_title/:collection_title/records/:id PUT Update record attributes
https://pine.io/:db_title/:collection_title/records/:id DELETE Remove a record
